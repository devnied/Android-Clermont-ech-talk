Type d’application payante

  • Achat in-App
  • Achat lors de l’installation
  • Publicité

In-App vs Achat lors de l’installation

  • Achat in-App
    • Développement nécessaire dans l’application
    • Seul Google et le développeur peuvent rembourser
    • Pas de catégorie distincte
  • Achat lors de l’installation
    • Gérer par le Play Store
    • 15 minutes pour se rétracter (2H pour les jeux)
    • Apparaît dans une catégorie distincte

Visibilité d’une application

  • Progression du nombre d’installation
  • Pourcentage de désinstallation
  • Catégorie
  • Note
  • Recherche
    • Nom de l’application
    • Description courte
    • Traduction

Note sur le store

  • Sans mise à jour la note d’une application diminue
  • Proposer au utilisateur de noter une application
  • Mise à jour fictive
  • Trouver des notes pour une application

Sécurité d’une application payante

  • Piratage
  • Vol d'applications
  • App licencing
    • Nécessite une permission
    • Mise en cache
  • Application à l'origine de l'installation

Conclusion
